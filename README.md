# 🐾 Stamppaw: 반려견 산책 문화 강화를 위한 웹앱 플랫폼

> Stamppaw는 반려견 보호자들이 더 안전하고, 더 편리하게, 그리고 더 즐겁게 산책 문화를 누릴 수 있도록 설계된 웹 기반 반려견 산책 플랫폼입니다.
>
> 개발 기간: 25.10.28 - 25.11.25

---

## 🎯 프로젝트 목표
- 안전한 반려견 산책 환경 제공  
- 보호자 간 소통 및 동행 기능을 통한 커뮤니티 강화  
- 산책 기록·미션·보상 시스템으로 지속적인 활동 유도  
- 반려견 용품 마켓까지 포함한 올인원 반려 생활 지원

## 🚀 주요 기능

### 1. 👤 회원 관리
- JWT 기반 회원가입 및 로그인  
- 이메일/닉네임 중복 검사  
- 프로필 조회 및 수정(닉네임, 이미지, 소개글)  
- 사용자 간 팔로우/언팔로우  
- 여러 마리 반려견 등록/조회/수정/삭제  

---

### 2. 🐕 산책 동행 시스템
- 동행 모집글 등록/조회/수정/삭제
- 채팅 버튼을 눌러 자동 채팅방 생성 → 사용자간 협의 
- 동행 신청 및 모집자의 수락/거절   
- 동행 종료 후 최대 5개 태그 기반 리뷰 작성  
- 작성/신청한 동행글 마이페이지에서 통합 관리  

---

### 3. 💼 반려견 산책 아르바이트
- 아르바이트 모집글 등록/수정/삭제  
- 지원자 목록 확인 및 수락/거절 처리  
- 채팅 기능 연동  
- 모달 기반 모집 상태(모집중/완료) 변경  
- 향후 결제·정산 및 신뢰도 시스템 확장 가능  

---

### 4. 🏃 산책 기록
- 실시간 위치 기반 산책 시작/종료  
- 이동 경로·거리·시간 추적  
- 사진 업로드 및 메모 작성  
- 산책 기록 상세보기 및 수정·삭제  

---

### 5. 🎖 포인트 · 미션 · 뱃지 시스템
- 산책·미션 달성 보상 포인트 제공  
- 랜덤 포인트 방문 시 보상  
- 산책 활동 기반 자동 뱃지 부여  
- 대표 뱃지 설정 기능  
- 관리자 페이지에서 미션·뱃지 관리 가능  

---

### 6. 🛍 마켓 & 커머스 기능
- 최신 상품·카테고리별 인기 상품 조회  
- 옵션/수량 선택 후 장바구니 담기  
- 장바구니 기반 주문/결제  
- TossPayments 연동 간편 결제 지원  
- 주문내역, 배송 상태 확인  
- 관리자 페이지에서 상품 및 주문/결제 관리
